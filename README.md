## Selected Work

### UATFlow Intelligence
A local UAT dashboard, tracker analyser, and evidence pack helper.

- Analyses Excel UAT trackers with flexible column mapping.
- Detects execution status across Pass, Fail, Blocked, No Run, Deferred, and N/A.
- Matches evidence files to test cases and supports structured reporting.

[View portfolio project](https://mnqobeey.netlify.app/projects)

### SauceDemo Automation Framework
BDD automation for a complete e-commerce checkout flow.

- Automates login, product selection, cart validation, checkout, and confirmation.
- Uses Gherkin scenarios, Cucumber step definitions, and Page Object Model locators.
- Includes reusable actions, assertions, waits, test data, Maven, and CI-ready structure.
